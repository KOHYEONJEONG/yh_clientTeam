# yh_clientTeam
##유한대학교 캡스톤 디자인 교수 프로그램(신희찬)

###작품설명
비대면 실시간 강의 시 학습을 도와주는 프로그램. 
로그인시 교수/학생 인식하여 다른 메인화면을 보여줌.
학생은 대표적으로 출석기능과 모르는 부분은 바로 질문할 수 있는 기능있어 수업에 집중하기 편함.
교수님은 대표적으로 스크린샷 요청하여 학생PC를 확인할 수 있고 수업을 이해했는지 문제를 낼 수 있어 관리하기 편함.
